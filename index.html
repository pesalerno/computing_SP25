COURSE SUMMARY: Biological data gets more expansive every year, making it crucial to develop and use computational tools to sort, organize, and analyze large datasets. The main goal of this course is to introduce the students to various tools that are available to them, so that they can independently work towards expertise. The course itself will not aim to make you experts, but rather to introduce you to many issues and tools to work around them. The first few weeks we will focus on unix/shell (computing basics) as well as on text editing (more important than people think!). We will then focus on python, and finally delve a bit into github and remote computing, all crucial tools for today's programmers. If time permits, we will also learn about bayesian statistics and apply our command-line knowledge to run some tree-inference analyses. We will end the course with independent projects, so that you all have a chance to apply the knowledge acquired on a subject of interest to you! Here is the link to the course syllabus




IMPORTANT: We will be working almost daily on hands-on coding, so always have a computer at hand (be sure to check one out from the College if in need).



BOOK AND RESOURCES: We will be mostly using the following resource for this course: Practical Computing for Biologists , by Haddock and Dunn. We will use several other resources but they will be provided as they are needed.
